Block pixel art for the mining block.

Add one PNG per block type, named exactly as the type id:
  stone.png, copper.png, iron.png, gold.png, platinum.png, mythril.png,
  runeStone.png, chaosFragment.png, gemstone.png, ancient.png,
  voidCrystal.png, shadowOre.png, dragonVein.png, astralCrystal.png,
  celestialCore.png

Recommended size for pixel art: 64×64 or 128×128. The game displays
them in a 160×160 box with object-fit: cover.

Test page: open test-block-art.html in the browser to preview.
